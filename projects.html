WPF Data Filtering Project

Programming Project

In this project, I developed a graphical user interface (GUI) to allow users to filter a data set and export the filtered data as a .json file in a WPF application. The project involved adding a csv file to the application and enabling users to select car colors from a ComboBox or choose 'All'. Additionally, users could enter a year and specify whether they wanted sales data that was equal to, less than or equal to, or greater than or equal to that year. Upon pressing a button, the application would filter the data based on user inputs and display the data in a list box. The GUI also included a button to export the list box contents to a json file. This project enhanced my skills in creating interactive and user-friendly applications, while also emphasizing the importance of data manipulation and export functionalities.
